💾 StorageNotes (SD Card Saver App)
Overview

This app allows users to write notes and save them into external storage (device storage) as a text file.

Features
Write and save notes
Stores data in external storage
Simple interface
Technologies Used
Java
XML
File Handling (Android)
Project Structure
StorageActivity.java → Handles file saving logic
activity_storage.xml → UI for note input
How It Works
User enters note
Clicks "Save"
Data is written to a file in external storage
Future Improvements
Add read/view notes feature
Add multiple file support
Add file manager integration
